# 🚀 ICMPTool

**ICMPTool** é uma ferramenta de reconhecimento de rede para Windows com interface gráfica, focada em aprendizado prático de **segurança de aplicações (AppSec)** e fundamentos de redes.

Desenvolvida em **C**, a aplicação oferece funcionalidades como *Ping*, *Traceroute* e *Scanner de rede*, permitindo entender na prática como protocolos de rede funcionam em baixo nível.

---

## 🎯 Objetivo do Projeto

Este projeto foi criado com foco educacional para:

- Aprender programação em **C aplicada a redes**
- Entender o funcionamento do protocolo **ICMP**
- Explorar conceitos de **network reconnaissance**
- Praticar fundamentos de **segurança da informação**
- Desenvolver interfaces gráficas nativas no Windows

---

## ⚙️ Funcionalidades

- ✅ Ping com exibição de tempo de resposta (RTT)
- ✅ Traceroute (Tracert) para análise de rotas
- ✅ Scanner de rede para descoberta de hosts ativos
- ✅ Interface gráfica simples e intuitiva
- ✅ Execução leve e rápida (sem dependências externas)
